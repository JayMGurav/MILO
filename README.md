
# My Drop dowwn

A fun drop down list with my details and nice little interactions

## Demo

![DropDown](demo.gif)

  
## Run Locally

Clone the project

```bash
  git clone https://github.com/JayMGurav/DropDown
```

Go to the project directory

```bash
  cd DropDown
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
